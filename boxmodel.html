Understanding the CSS Box Model

This is a `div` element with content, padding, border, and margin applied.

The total width of the box is calculated as: 200px (width) + 40px (padding) + 10px (border) + 60px (margin) = 310px

The total height of the box is calculated as: 100px (height) + 40px (padding) + 10px (border) + 60px (margin) = 210px
