Using Different Types of Positioning in CSS

Static Position

This box is statically positioned. It follows the normal document flow.

Relative Position

This box is relatively positioned. It is moved 20px down and to the right from its original position.

Absolute Position

This box is absolutely positioned. It is placed 100px from the top and 50px from the left of its nearest positioned ancestor or the viewport.

Fixed Position

This box is fixed to the top right of the viewport. It stays there even when the page is scrolled.

Sticky Position

This box is sticky. It starts relative, but sticks to the top of the viewport as you scroll past it.
